# MercadoLibre Scraper 

Scraper hecho con Puppeteer que obtiene productos desde MercadoLibre y los exporta a Excel (Lo utilice para Iphones en Argentina)

##  Features 
- Scraping dinámico y actualizados (selectores nuevos + fallback)
- Soporte múltiples modelos
- Exportación a Excel con resumen
- Anti cambios de DOM

##  Tecnologías 
- Node.js
- Puppeteer
- ExcelJS

##  Uso (Terminal) 

npm install  
node scraper.js

## Utilidad 
- Análisis de competencia
- Tracking de precios
- Detectar oportunidades de reventa
